3D 产品展示工具使用说明

项目地址：
https://github.com/asuka091241-ai/product-viewer-builder

一、怎么打开工具

推荐使用桌面快捷方式“产品展示工具”打开。

如果没有快捷方式，也可以在本文件夹里启动本地服务：

python -m http.server 5182 --bind 127.0.0.1

然后用浏览器打开：

http://127.0.0.1:5182/


二、基本流程

1. 打开工具页面。
2. 点击中间的添加模型，导入 OBJ / GLB / GLTF 模型。
3. 调整材质、灯光、HDRI、部件位置和展开效果。
4. 点击“结果预览”检查客户最终看到的效果。
5. 根据需要选择导出方式。


三、三个导出/预览按钮怎么用

1. 结果预览

只是在浏览器里临时打开预览页，用来检查效果。
这个不是最终交付文件。

2. 生成 HTML 文件

会生成一个单独的 .html 文件。
客户只需要拿到这一个 HTML 文件，双击就能看。

如果只是发给客户查看，优先用这个。

3. 生成展示网页 ZIP

会生成一个完整展示网页压缩包。
解压后里面有 open-directly.html 和 index.html。

open-directly.html：
适合直接发客户，客户双击就能看。

index.html：
适合上传到网站、服务器、对象存储或静态托管平台。
不要只单独发 index.html，它需要同文件夹里的 viewer.js、styles.css、config.js 和 assets 文件夹。


四、发客户推荐

最简单：
发“生成 HTML 文件”得到的单个 HTML。

也可以：
发 ZIP 展示包里的 open-directly.html。

如果客户要放到网站：
发完整 ZIP 展示包，解压后上传整个文件夹，访问 index.html。


五、安全提醒

导出的文件会把模型转换成网页展示数据，客户不会直接拿到原始 OBJ / GLB / GLTF 文件。

但网页端 3D 最终必须在浏览器里解密和渲染，所以不是绝对防逆向。
对外展示时建议使用展示版模型，不要放生产级源模型或敏感内部结构。
